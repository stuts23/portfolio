PlayCTF

  • Category: User Research, Rapid Prototyping, QA Testing
  • Project Duration: March 2021 - June 2022
  • Project URL: playCTF
  • Documentation URL: Coming Soon

PlayCTF is a Jeopardy Style CTF hosting service that helps manage deployment, lifecycle and health check of CTFs. I worked on admin panel with features to configure CTFs, deploy and manage challenges and view competition statistics in an accessible form. Even designed homepage with attention mapping tools for best conversion rates.
